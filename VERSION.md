# LeHalo Version Information

## Current Version: 2.0.0

**Release Date:** January 28, 2025

## Version History

### v2.0.0 (Current)
- **Critical Bug Fixes:**
  - Fixed binary data corruption in request body parser
  - Fixed hanging connections in catch-all route
  - Fixed TypeError with undefined baseUrlObj in URL rewriting
- **All features from v1.0.2**

### v1.0.2
- **Major Enhancements:**
  - Universal website support (YouTube, Twitch, ChatGPT, Character.ai, Netflix, Walmart, etc.)
  - Comprehensive URL rewriting (absolute, relative, protocol-relative)
  - JavaScript fetch/XMLHttpRequest interception
  - Dynamic proxy script injection
  - Enhanced iframe permissions
  - Catch-all route for relative paths
  - Improved error handling

### v1.0.1
- Tabbed browser interface
- Bookmarks system
- Browsing history
- Settings panel
- Homepage integration
- Logo throughout site

### v1.0.0
- Initial release
- Core proxy functionality
- Homepage with search
- Funny text randomizer
- Basic proxy services

## Compatibility

- **Node.js:** >= 18.0.0
- **Browsers:** Chrome, Firefox, Safari, Edge (latest versions)
- **Platforms:** Windows, macOS, Linux

## Known Limitations

- DRM-protected content (Netflix, some YouTube videos) may have limited functionality
- WebSocket connections are converted to HTTP (basic support)
- Some advanced JavaScript features may require additional configuration

## Upgrade Notes

### From v1.0.2 to v2.0.0
- No breaking changes
- Critical bug fixes for production stability
- Improved binary data handling
- Better error handling

### From v1.0.1 to v2.0.0
- No breaking changes
- Enhanced proxy capabilities
- Better website compatibility
- Critical bug fixes

### From v1.0.0 to v2.0.0
- New tabbed browser interface
- Bookmarks and history features
- Enhanced proxy with universal website support
